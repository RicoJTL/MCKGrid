## Packages
framer-motion | Smooth page transitions and UI animations
recharts | Visualizing race statistics and championship standings
date-fns | Formatting race dates and times
clsx | Utility for conditional class names
tailwind-merge | Utility for merging tailwind classes efficiently

## Notes
Auth is handled via Replit Auth and provided `useAuth` hook.
No client-side routing for login needed - redirect to `/api/login`.
Using Lucide React for all icons.
Mobile-first approach using a sidebar layout that collapses on mobile.
